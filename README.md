## Description
This project implements a notification system that sends and receives notifications when users upload photos or videos in a mobile application.

## Features
- Upload photos and videos
- Send notification when upload starts or completes
- Receive notifications related to media uploads
- Simple and user-friendly workflow

## Technologies Used
- Mobile application framework
- Notification service
- Media upload handling

## How It Works
When a user uploads a photo or video in the app, the system triggers a notification to inform relevant users or the system about the upload status.

## Purpose
This project was developed as **Firebase Project** to demonstrate handling notifications during media upload in a mobile application.
